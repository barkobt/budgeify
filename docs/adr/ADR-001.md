# ADR-001: Centralized Logger Utility

**Status:** Accepted
**Date:** 2026-02-06
**Context:** Production code contained 14 raw `console.error` / `console.log` calls. No structured logging, no environment-aware suppression.

**Decision:** Create `src/lib/logger.ts` with level-based logging (debug/info/warn/error). In production, suppress debug and info. All modules must use `logger` instead of raw console.

**Consequences:**
- Cleaner production output
- Structured log messages with context prefix
- Easy to extend with remote logging later
