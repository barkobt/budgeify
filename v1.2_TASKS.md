## ðŸŽ¨ Design Decisions Log

### Color Choices
**Rationale:** Cosmic Indigo palette chosen for:
1. Deeper contrast than v1.1
2. More "space-age" feel
3. Better OLED black performance
4. Reduced eye strain in dark environments

### Font Choice: Inter
**Rationale:**
1. Most readable sans-serif for UI
2. Variable font = better performance
3. 9 weights for perfect hierarchy
4. Open source, free to use

### AI Button Position
**Rationale:** Bottom-right corner because:
1. Thumb-friendly on mobile
2. Doesn't obstruct content
3. Standard chat widget position
4. Easy to discover
